### Was sind Variablen?

Variablen sind Platzhalter für dynamische Werte, die während der Nutzung eines Formulars generiert werden.

Diese dynamischen Werte erlauben es, maßgeschneiderte Texte und Dokumente basierend auf den individuellen Eingaben der Nutzer\:innen zu erstellen.

### Woher kommen Variablen?

Jede Nutzer\:inneneingabe in Eingabe-Blöcken erzeugt eine Variable. Die einzige Ausnahme hier ist zurzeit die Multi-Select Eingabe. Diese kann nur innerhalb von Dokumenten eingebunden werden.

Außerdem erzeugen Berechnungs-Blöcke das Ergebnis ihrer Berechnung als Variable.

### Variablentypen

Es gibt verschiedene Variablentypen. Dies ist entscheidend für die Bedingungen, da die Erstellung des Bedingungssatzes von der Art der Variable abhängt. Folgende Typen gibt es:

* Einfachauswahl: Diese Variable wird durch die Eingabe "Einfachauswahl" produziert.
* Mehrfachauswahl: Diese Variable wird durch eine Eingabe "Mehrfachauswahl" produziert.
* Text: Diese Variable wird durch die Eingaben "Texteingabe", oder "Textfeld" produziert.
* Zahl: Diese Variable wird durch die Eingabe "Zahleneingabe" produziert.
* Block-Status: Diese Variable wird durch den Status eines Blocks produziert.

Jeder Variablentyp hat ein eigenes Icon, das an der Variable angezeigt wird. Die Variable des Blocks, also der Blockstatus, behält das Icon des Blocktyps.

### Einsatz von Variablen in deiner Anwendung

#### Integration von Variablen in Dokumente

Wie Variablen in Dokumenten eingebunden werden, erfährst du hier: [Dynamische Dokumente](/docs/dokumente "Dynamische Dokumente")

#### Integration von Variablen in den Texteditor

Variablen können in den Texteditor von Informations-, Report- und Eingabe-Blöcken eingebunden werden.

Wähle das @-Symbol auf deiner Tastatur, um eine Dropdown-Liste mit verfügbaren Variablen zu öffnen.

Wähle die gewünschte Variable aus der Liste aus und sie wird in deinen Text integriert.
