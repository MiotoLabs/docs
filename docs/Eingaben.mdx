---
title: Eingaben
---

Eingaben fragen Daten ab. Jede Eingabe produziert eine Variable - einen unbekannten Wert. Dieser Wert kann in Bedingungen eingesetzt werden und dadurch den Verlauf deiner Anwendung steuern. Außerdem kann der Wert als Variable im Formular oder in deinen dynamischen Dokumenten eingesetzt werden. Aber bevor wir dazu kommen, erst einmal zurück zu den Basics.

### Eingabetypen

* **Einfachauswahl:** Ermöglicht die Auswahl einer Antwort aus mehreren vordefinierten Antworten.
* **Mehrfachauswahl:** Ermöglicht die Auswahl einer Antwort aus mehreren vordefinierten Antworten.
  * Aufgrund der komplexen Logik dahinter, können Mehrfachauswahl-Eingaben derzeit nur im Dokumenten-Template verwendet werden. Sie können im Builder nicht als Variable eingebunden werden oder die Verbindung determinieren.
* **Texteingabe:** Eingabe einer Textzeile.
* **Textfeld:** Eingabe mehrerer Textzeilen.
* **Nummerneingabe:** Eingabe einer Zahl aus einem vordefinierten Zahlenbereich.

### Eingaben hinzufügen

Klicke im Block-Editor auf den Button "Eingabe hinzufügen". Anschließend kannst du aus allen Eingabetypen wählen.

### Inhalt der Eingaben

Eingaben werden im Block-Editor als Karten dargestellt. Die Eingabekarten haben verschiedene Bearbeitungsmöglichkeiten.

#### Eingabemenü

Jede Eingabekarte hat ein Dreipunkt-Menü. In diesem Menü kann:

* Die Word-Variable zur Erstellung dynamischer Dokumente herauskopiert werden.
* Der Eingabetyp geändert werden.
* Die Eingabe gelöscht werden.

#### Eingabename

Der Eingabename legt fest, wie die Eingabe referenziert wird. Bei der Variablenauswahl werden Blöcke mit ihren Eingaben in einem Dropdown dargestellt. Eingaben werden dort anhand ihres Namens identifizierbar.

#### Eingabelabel

Lege das Label fest, das im Formular über der Eingabe stehen soll. Standardmäßig ist das Label aktiviert - du kannst es aber auch deaktivieren. Hat ein aktives Label keinen Inhalt, übernimmt es den Inhalt aus dem Eingabenamen.

#### Pflichtfeld

Standardmäßig ist eine Eingabe als Pflichtfeld markiert. In der Eingabekarte kannst du festlegen, ob die Eingabe ein Pflichtfeld sein soll, oder nicht.

#### Nur bei Ein- und Mehrfachauswahl: Antwortoptionen definieren

Beim Eingabetyp "Einfachauswahl" und "Mehrfachauswahl" kannst du in der Eingabekarte die Antwortauswahl hinzufügen. Klicke dafür auf "Antwort hinzufügen" und definiere die mögliche Antwort. Mit dem Handle-Icon kannst du die Reihenfolge der Antworten festlegen.

#### Reihenfolge der Eingaben ändern

Klicke auf den kleinen Button "Reihenfolge ändern" links neben dem Button "Eingabe hinzufügen". Anschließend schließen sich die Eingabekarten und du kannst sie per drag-and-drop an die gewünschte Stelle verschieben.
