Öffnest du eine Anwendung, öffnet sich ihr Builder.

### Builder-Interface

Das Builder-Interface besteht im wesentlichen aus dem *Canvas*, auf dem du deine Mioto-Anwendungen erstellst.

Auf dem Canvas fügst du Blöcke hinzu und verbindest diese miteinander, um deiner Anwendung die gewünschte Funktionalität zu geben.

### Blöcke

Blöcke sind die elementaren Bausteine deiner Anwendung. Jeder Block repräsentiert eine spezifische Aktion, die deine Anwendung ausführen kann.

Die grundlegenden Aktionen von Blöcken umfassen das Stellen von Fragen, das Sammeln und Austauschen von Daten sowie das Generieren von Dokumenten.

### Start- und Endblock

Jede Anwendung beginnt mit einem Startblock, der durch ein Raketen-Icon gekennzeichnet ist. Von hier aus entfaltet sich der Entscheidungsbaum deiner Anwendung.

Deine Anwendung endet mit einem oder mehreren Endblöcken. Diese verhindern, dass in der fertigen Anwendung ein *zum nächsten Schritt*-Button erscheint. Endblöcke werden im Blockmenü festgelegt.

### Block-Konfiguration und Verzweigung

Jeder Block hat einen zugehörigen Editor, der geöffnet wird, indem du auf den Block klickst. Hier kannst du spezifische Einstellungen für den Block vornehmen und seine Verbindungen definieren.

Jeder Block kann sich verzweigen, das heißt, er kann mehr als eine Ausgangsverbindung haben. Dies ermöglicht die Erstellung eines Entscheidungsbaums in deiner Anwendung.

Die Bedingungen für die Aktivierung einer Verbindung werden im Block-Editor festgelegt. Du kannst neue Verbindungen erstellen, für jede Verbindung eine Bedingung und ein Ziel definieren.

Bedingungen bestehen aus einem Eingabewert, einem Vergleichsoperator und einem Vergleichswert. Der Eingabewert muss aus einer Eingabe innerhalb eines Blocks stammen, aber du kannst alle existierenden Eingaben abrufen.

Bedingungen können sehr komplex sein, indem du verschiedene Bedingungssätze miteinander verknüpfst. Du kannst beispielsweise zwei Bedingungen erstellen und zwischen ihnen die Option *Und* wählen, um eine komplexere Bedingung zu erstellen. Mit der Option *Oder* kannst du zwei Bedingungen unabhängig voneinander für eine Verbindung geltend machen.
